Properties

Basic Properties

this.prop.$id Type: String

The id of the DOM object rendered as the designer; this property cannot be empty, and the designer that needs to be rendered has the id attribute on it.

this.$refs.$tool Type: Vue Component

The object on the left side of the designer's toolbar, which is valid only when the parameter is initialized with property.haveTool=true; otherwise the property is undefined.

this.$refs.$head Type: Vue Component

The object ofthe designer's headbar, including the title box and the top button group; it is valid only when the argument is initialized with property.haveHead=true; otherwise the property is undefined.

this.$refs.$workArea Type: Dom Object

Load the node/line/group lane drawing workspace.

this.$refs.$draw Type: Dom Object

Paint a vector line of Dom containers in the workspace.

this.$refs.$group Type: Dom Object

A Dom object container that is only used to load grouped swimlane DOM elements in the work area; it is valid only when the argument is initialized with property.haveGroup=true; otherwise the property is undefined.

this.$refs.$paper Type: Dom Object

A Dom object container that is only used to load memos DOM elements in the work area; it is valid only when the argument is initialized with property.haveMemo=true; otherwise the property is undefined.

this.prop.$nodeCustoms Type: JsonMap

The mapping configuration that we only use the custom background map to display for some special node types.
The key of Json is the name of the node type, and value is also a Json structure：
{ class : 'The style class of the node in addition to custom background graph', width : 'The fixed width of the node', height : 'The fixed height of the node' }

this.prop.$slotNodeRenders Type: JsonMap

The rendering configuration that only uses custom html label templates is adopted for the slot type nodes, (Since version 1.5)
The key of Json is the name of the node type, and value is also a Json structure：
{ minWidth : 'Minimum width', minHeight : 'Minimum height', resizable:Whether resizing is supported, initInfo:{A blank slot defination} }

this.prop.$workExtendStep Type: Number

When expanding the available area of the drawing workspace manually, how many pixels are added after expanding/width at a time, the default is 200px.

this.$refs.$ghost Type: Vue Component

A translucent floating area that is used by the user to operate on elements such as moving and resizing. Only if this.prop.$editable=true the property is only present when.

this.$refs.$mpFrom Type: Vue Component

After the line is selected in edit mode, the small box at the end point arrow is used to move to change the arrival node to be connected. Only if this.prop.$editable=true the property is only present when.

this.$refs.$mpTo Type: Vue Component

After the line is selected in edit mode, the small box at the end point arrow is used to move to change the arrival node to be connected. Only if this.prop.$editable=true the property is only present when.

this.$refs.$textArea Type: Vue Component

The floating text field that appears after double-clicking an element in edit mode is used to write setName the new name pass arguments required by the rename method . Only if this.prop.$editable=true the property is only present when.

this.$refs.$lineMove Type: Vue Component

The floating Div used when manipulating the midline of a moving polyline. Only if this.prop.$editable=true the property is only present when.

this.$refs.$lineOper Type: Vue Component

The floating Div used when manipulating the midline of a moving polyline. Only if this.prop.$editable=true the property is only present when.

this.prop.$navMap Type: Dom Object

Thumbnail navigation, default is undefined. This property only exists if the GooFlow component uses the navmap.js extension package and successfully passes the navSelector pass parameter to specify the correct render dom object.

Business Properties

this.prop.$nowType Type: String

The current type of object to be drawn starts with "cursor": no element is drawn, but is selected as a mouse pointer.
Other values include: "direct" solid lines, "dashed" dotted lines, "group" grouping lanes, …… various types of nodes.

this.prop.$focus Type: String

The currently selected node/conversion line ID, if not selected or the workspace is empty, it will be an empty string.

this.prop.$editable Type: Boolean

Whether the current workspace is editable, that is, edit mode or browse mode only.

this.prop.$dragAddOper Type: Boolean

The current way to add block elements is to use drag-and-drop mode when true and click mode when false; the default is false.

this.scale Type: Number

The scale of the flowchart in the workspace is currently relative to the original size. The default is 1 (100%) and the range is 0.5~4.

this.prop.$headBtnEvents Type: JsonMap

Users of the head of field additionally customize the type of event button bindings map collections to button type name as a key, corresponding function type variableto value.

this.prop.$undoStack Type: Array[Json Object]

The transaction stack of "undo"; only when initialization parameter passing property.haveTool=true and property.useOperStack=true when the property is only present.

this.prop.$redoStack Type: Array[Json Object]

The transaction stack of "redo"; only when initialization parameter passing property.haveTool=true and property.useOperStack=true when the property is only present.

this.events Type: JsonMap

Built-in collection of all event method interfaces; with < code > event name as key, corresponding < code > function type variable as value.

this.prop.$multiKey Type: String

When the mouse clicks to select an element in the flowchart, at the same time it holds down which function keys to achieve multi-selection. There are < code > shift , < code > Ctrl two values, default is shift key.

Data-store Properties

this.prop.$title Type: String

The name of the flowchart to be loaded; if there is a top bar and a title bar when it is initialized, it is displayed in the title box.

this.prop.$nodes Type: JsonMap

Loading of all the nodes in the flowchart Map data set; to node's idas a key, NodeItem data type structure to value.
For details, see "NodeItem" in the "Data Definition" chapter.

this.prop.$nodeCount Type: Number

The number of nodes in the loaded flowchart, initially 0.

this.prop.$lines Type: JsonMap

Loading of all the lines in the flowchart Map data set; to line's idas a key, LineItem data type structure to value.
For details, see "LineItem" in the "Data Definition" chapter.

this.prop.$lines Type: Number

The number of lines in the loaded flowchart, initially 0.

this.prop.$areas Type: JsonMap

Loading of all the grouping lanes in the flowchart Map data set; to lane's idas a key, AreaItem data type structure to value.
For details, see "AreaItem" in the "Data Definition" chapter.

this.prop.$areaCount Type: Number

The number of grouped lanes in the loaded flowchart, initially 0.

this.prop.$memos Type: JsonMap

Loading of all the memos in the flowchart Map data set; to memo's idas a key, MemoItem data type structure to value.
For details, see "MemoItem" in the "Data Definition" chapter.

this.prop.$memoCount Type: Number

The number of memos in the loaded flowchart, initially 0.

this.prop.$max Type: Number

Calculate the SEQUENCE of the suffix sequence number in the default element name. The default value is 1;
the initial value comes from the initNum attribute of the parameters passed in the constructor, or the initNum attribute of the flowchart JSON data is loaded;
when there are elements in the flowchart, When manually added, the value increases by one.

this.prop.$deletedItem Type: JsonMap( {id: String )

It is deleted in the editing operation in the flowchart id-map element set;
to element's idas key，element category name as value(There are four values: node', 'line', 'area', 'memo').

this.prop.$extra Type: Extra

The additional data map set in the loaded flowchart does not directly participate in the construction of the flowchart, and the output of the flowchart data will be returned as it is. For details, see "Extra" in the "Data Definition" chapter.
