Constructors

Methods

There are two constructor calls, their params are the same, and their return value is also the same.

    var demo = GooFlow.init(selector, options);    //Factory mode
    var demo = new GooFlow(selector, options);     // native definition of class object method

Personally, it is recommended to use the first (factory mode), compatible with the modular front-end development mode:

GooFlow.init('selector', [options])

Parameters

selector String

The Dom container selector to be rendered as a designer, in the same format as the Function of document.querySelector's params, the dom requires an id attribute.
